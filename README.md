<h3>📘 Overview</h3>
<p>
<em>AyurSutra</em> is a full-stack web platform that modernizes the management of <b>Panchakarma therapies</b> — combining ancient Ayurvedic wisdom with modern technology. It offers digital patient records, smart therapy scheduling, and personalized recovery tracking to make Ayurveda more transparent, data-driven, and globally scalable.
</p>

<p>
The platform empowers patients, practitioners, and administrators with an integrated system for <strong>scheduling, notifications, teleconsultation,</strong> and <strong>real-time feedback analysis</strong>, bringing technology and tradition together.
</p>

<h2>🧩 Problem Statement</h2>

<ul>
  <li>Manual scheduling and double bookings causing inefficiency.</li>
  <li>Lack of centralized digital patient records.</li>
  <li>Missed therapy instructions and poor patient adherence.</li>
  <li>No feedback-driven personalization or standardization.</li>
  <li>Inconsistent care quality across Panchakarma centers.</li>
</ul>

<h2>💡 Our Solution</h2>

<ul>
  <li><strong>Smart Scheduler:</strong> Conflict-free therapy slot assignment based on availability and duration.</li>
  <li><strong>Digital Patient Records:</strong> Centralized therapy history and recovery tracking.</li>
  <li><strong>Smart Notifications:</strong> Email and in-app reminders for pre- and post-procedure care.</li>
  <li><strong>Real-Time Therapy Tracking:</strong> Visual dashboards showing treatment milestones.</li>
  <li><strong>Feedback Personalization:</strong> Uses patient feedback to improve future therapy planning.</li>
  <li><strong>Standardized Protocols:</strong> Ensures consistency across treatment centers.</li>
  <li><strong>Virtual Consultations:</strong> Enables remote accessibility for patients.</li>
  <li><strong>Gamified Wellness Challenges:</strong> Encourages healthy habits through daily challenges.</li>
</ul>

<h2>⚙️ Installation & Setup</h2>

<ol>
<li>Clone the repository:
<code>git clone https://github.com/balyanankit16-code/AyurSutra__.git</code></li>

<li>Navigate to the Backend folder and install dependencies:
<code>npm install</code></li>

<li>Navigate to the Frontend folder and install dependencies:
<code>npm install</code></li>

<li>Create a <code>.env</code> file with your MongoDB URI, JWT Secret and Email API credentials.</li>

<li>Start Backend:
<code>npm run dev</code></li>

<li>Start Frontend:
<code>npm run dev</code></li>

</ol>
