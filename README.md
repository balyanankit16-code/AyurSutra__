<h3>📘 Overview</h3>
<p>
<em>AyurSutra</em> is a full-stack web platform that modernizes the management of <b>Panchakarma therapies</b> — combining ancient Ayurvedic wisdom with efficiency. 
It offers digital patient records, smart therapy scheduling, and personalized recovery tracking to make Ayurveda more transparent, data-driven, and globally scalable.
</p>
<p>
The platform empowers patients, practitioners, and administrators with an integrated system for <strong>scheduling, notifications, teleconsultation,</strong> and <strong>real-time feedback analysis</strong>—bringing technology and tradition together.
</p>

<h2>🧩 Problem Statement</h2>
<ul>
  <li>The Panchakarma process today faces major challenges such as:</li>
  <ul>
    <li>Manual scheduling and double bookings causing inefficiency</li>
    <li>Lack of centralized digital patient records</li>
    <li>Missed therapy instructions and poor adherence</li>
    <li>No feedback-driven personalization or standardization</li>
    <li>Inconsistent care quality across centers</li>
  </ul>
</ul>

<h2>💡 Our Solution</h2>
<ul>
  <li><strong>Smart Scheduler:</strong> Conflict-free therapy slot assignment based on availability and duration.</li>
  <li><strong>Digital Patient Records:</strong> Centralized therapy data and progress tracking.</li>
  <li><strong>Smart Notifications:</strong> Pre- & post-procedure reminders via SMS, email, and in-app alerts.</li>
  <li><strong>Real-Time Therapy Tracking:</strong> Visual dashboards with milestones for patients and doctors.</li>
  <li><strong>Feedback Personalization:</strong>  patient for better understanding.</li>
  <li><strong>Standardized Protocols:</strong> Ensures quality and consistency across centers.</li>
  <li><strong>Virtual Consultations:</strong> Enables remote and rural accessibility.</li>
  <li><strong>Gamified Wellness Challenges:</strong> Daily yoga, breathing, and lifestyle streaks for motivation.</li>
</ul>

<section>
  <h2>🧱 Technology Stack</h2>

  <h3>Frontend</h3>
  <ul>
    <li><strong>Framework:</strong> React.js – Dynamic, component-based UI.</li>
    <li><strong>Styling:</strong> Tailwind CSS – Modern responsive layouts.</li>
    <li><strong>Data Visualization:</strong> Chart.js / Recharts – Therapy progress charts.</li>
    <li><strong>Routing:</strong> React Router – Seamless navigation.</li>
  </ul>

  <h3>Backend</h3>
  <ul>
    <li><strong>Runtime:</strong> Node.js – Server-side JavaScript runtime.</li>
    <li><strong>Framework:</strong> Express.js – RESTful APIs & routing.</li>
    <li><strong>Authentication:</strong> JWT + bcrypt – Secure role-based access control.</li>
    <li><strong>Notification Service:</strong>Email APIs – Smart reminders.</li>
  </ul>

  <h3>Database</h3>
  <ul>
    <li><strong>MongoDB:</strong> For patient records, practitioners, therapy sessions, and feedback data.</li>
  </ul>

<h2>⚙️ Installation & Setup</h2>
<ol>
  <li>Clone the repository: <code>git clone https://github.com/yourusername/AyurSutra.git</code></li>
  <li>Navigate to the backend folder and install dependencies: <code>npm install</code></li>
  <li>Navigate to the frontend folder and install dependencies: <code>npm install</code></li>
  <li>Create a <code>.env</code> file with database URIs, JWT secret, and API keys (Twilio/Firebase).</li>
  <li>Start backend server: <code>npm run dev</code></li>
  <li>Start frontend: <code>npm run dev</code></li>
</ol>

<h2>📡 API Endpoints</h2>
<ul>
  <li><code>POST /auth/register</code> – Register user (Patient, Practitioner, Admin)</li>
  <li><code>POST /auth/login</code> – Login & retrieve JWT token</li>
  <li><code>GET /therapy/schedule</code> – Fetch upcoming therapy sessions</li>
  <li><code>POST /feedback/submit</code> – Add feedback after therapy</li>
  <li><code>GET /analytics/progress</code> – Retrieve progress dashboard data</li>
